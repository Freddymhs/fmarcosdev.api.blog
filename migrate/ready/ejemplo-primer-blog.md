# Mi Primer Blog

Este es un ejemplo de cómo se verían tus archivos `.md`.

## ¿Qué es esto?

Este archivo es solo un **ejemplo** para que pruebes la migración.

### Características

- Soporta **markdown** completo
- Puedes usar `código inline`
- Listas numeradas y con viñetas
- Y mucho más!

```javascript
// Incluso bloques de código
const ejemplo = "Hola mundo";
console.log(ejemplo);
```

## Cómo usar

1. Borra este archivo
2. Coloca tus propios `.md` aquí en la carpeta `source/`
3. Mueve los que quieras subir a `ready/`
4. Ejecuta el script de migración

¡Listo! 🚀
